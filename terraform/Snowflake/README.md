# Prerequisits:
- snowflake account (required)
- terraform installed

# Get Started
**To create the snowflake infrastructure for ansible [automation][1] or local tests:**
1. fill in the values in terraform.tfvars
2. run the following commands:
   ```
    terraform init

    terraform plan
    
    terraform apply
   ```
